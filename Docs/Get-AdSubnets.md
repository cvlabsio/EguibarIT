# Get-AdSubnets

## SYNOPSIS
Get AD subnets from current Forest

## SYNTAX

```
Get-AdSubnets [<CommonParameters>]
```

## DESCRIPTION
Reads all Subnets from the current Forest and store those on an array.

## EXAMPLES

### EXAMPLE 1
```
Get-AdSubnets
```
