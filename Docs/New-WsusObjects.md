# New-WsusObjects

## SYNOPSIS
Create WSUS Objects and Delegations

## SYNTAX

```
New-WsusObjects [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Create the WSUS Objects used to manage
this organization by following the defined Delegation Model.

## EXAMPLES

### EXAMPLE 1
```
New-WsusObjects
```
